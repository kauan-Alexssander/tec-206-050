HTML E PRA QUE SERVE

HTML (HyperText Markup Language, ou Linguagem de Marcação de Hipertexto) é a base de quase tudo o que você vê na internet. Ele não é uma linguagem de programação, mas sim uma linguagem de marcação. Isso significa que ele serve para estruturar e dar sentido ao conteúdo de uma página web, dizendo ao navegador o que é um título, o que é um parágrafo e onde está uma imagem. Para entender o HTML de forma simples, pense nele como o esqueleto de um site. Se fôssemos fazer uma analogia com a construção de uma casa, teríamos a seguinte divisão: HTML: São os tijolos e as colunas (a estrutura).
